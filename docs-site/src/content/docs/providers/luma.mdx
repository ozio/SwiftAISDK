## Notes

Luma images mirror upstream Luma 3.0.6: `LUMA_API_KEY`, bearer auth, custom base URL and headers, versioned user-agent suffix, `image`/`imageModel` aliases, `luma.image` provider IDs, `photon-1` and `photon-flash-1` model IDs, `/dream-machine/v1/generations/image`, async polling, unauthenticated generated image download, warnings for unsupported size/seed, `providerOptions.luma` referenceType/images/polling schema, URL-only image references, response metadata, and unsupported language/embedding families.
